주의 사항

매크로란?

매크로란 자주 반복하는 일련의 과정을 한 번에 처리하도록 만드는 기능입니다. 매크로를 이용하면 여러 개의 스킬이나 명령어를 아이콘 하나로 쉽게 사용할 수 있습니다.

매크로 실행하기

<시작 메뉴 - 매크로>를 클릭해 매크로 편집 창을 엽니다. 매크로 목록에서 원하는 매크로 아이콘을 퀵바에 등록하여 실행하거나, 매크로 명령어를 채팅 창에 직접 입력하여 실행합니다.

목록으로

매크로 등록하기

매크로 창의 매크로 탭에서 '새 매크로' 버튼을 눌러 새 매크로를 등록할 수 있습니다. 최대 12개까지 등록할 수 있습니다.

매크로 삭제하기

매크로 창에서 삭제할 매크로 아이콘을 선택하고 '삭제' 버튼을 클릭합니다.

매크로 수정하기

1. 수정할 매크로 아이콘을 선택하고 '수정' 버튼을 클릭합니다.

2. 매크로 편집 창에서 매크로 구문을 편집한 뒤 '저장' 버튼을 클릭합니다.

목록으로

매크로 편집 창의 구성

매크로 이름: 매크로를 퀵바에 등록해 사용하면 매크로 이름에 입력한 내용이 툴팁으로 표시됩니다. 띄어쓰기를 포함해 최대 10자까지 입력할 수 있습니다.

매크로 설명: 해당 매크로의 기능에 대해 간략하게 적습니다. 띄어쓰기를 포함해 최대 40자까지 입력할 수 있습니다.

아이콘 선택: 매크로 아이콘으로 사용할 수 있는 아이콘입니다.

매크로 명령 입력: 매크로 기능을 구현할 명령어를 입력합니다. 띄어쓰기를 포함해 최대 255자까지 입력할 수 있습니다.

매크로 자동 입력

매크로 편집 창의 매크로 명령 입력란에 매크로에 등록할 스킬, 기능, 감정 표현, 퀵바, 아이템 아이콘을 끌어다 놓으면 자동으로 명령어가 입력됩니다.

매크로 수동 입력 - 스킬

매크로 편집 창의 매크로 명령 입력란에 '/스킬 명령어 [스킬 이름]'을 입력합니다. 스킬 이름을 띄어쓰기까지 정확히 입력해야 매크로가 제대로 구현됩니다.

예) /스킬 맹렬한 일격 II

매크로 수동 입력 - 기능

매크로 편집 창의 매크로 명령 입력란에 '/스킬 [기능 이름]'을 입력합니다. 기능 이름을 정확히 입력해야 매크로가 제대로 구현됩니다.

예) /스킬 아이템 줍기

매크로 수동 입력 - 감정 표현

매크로 편집 창의 매크로 명령 입력란에 '/[감정 이름]'을 입력합니다.

예) /웃기

매크로 수동 입력 - 퀵바

매크로 편집 창의 매크로 명령 입력란에 '/퀵바 [퀵바 종류] [퀵바 줄수] [퀵바 번호]'를 입력합니다. 퀵바 종류는 기본 퀵바(1), Alt 퀵바(2), Ctrl 퀵바(3), 오른쪽 퀵바(4) 순입니다.

예) /퀵바 1 1 1 - 기본 퀵바의 1째 줄의 1번 슬롯에 등록된 아이콘을 실행합니다.

매크로 수동 입력 - 아이템

매크로 편집 창의 매크로 명령 입력란에 '/사용 [아이템 이름]'을 정확히 입력합니다.

예) /사용 용병의 수통

매크로 수동 입력 - 대상 선택

매크로 편집 창의 매크로 명령 입력란에 '/선택 [캐릭터명]'을 입력합니다. 캐릭터 이름 대신 [%자신], [%펫], [%파티0]~[%파티5], [%펫1~5], [%대상], [%이전대상], [%대상의대상]을 입력할 수도 있습니다.

예) /선택 [%펫1] - 1번 파티원의 펫을 선택합니다.

/선택 [%이전대상] - 이전에 선택했던 대상을 선택합니다.

매크로 수동 입력 - 채팅

매크로 편집 창의 매크로 명령 입력란에 '/[채팅모드] [내용]'을 입력합니다.

예)/귓속말 멋쟁이 안녕하세요

채팅 내용 중간에 [%자신], [%펫], [%파티0~5], [%펫1~5], [%대상], [%이전대상], [%대상의대상]도 입력할 수 있습니다.

예) /파티 역시 [%파티1] 님 최고!

매크로 수동 입력 - 변수

매크로 편집 창의 매크로 명령 입력란에 '/변수 [숫자0~9] [이름]'을 입력해 자주 사용하는 이름을 변수로 지정할 수 있습니다.

예) /변수 0 최하급 생명의 물약

긴 이름을 변수로 등록하면 매크로를 간단하게 작성할 수 있습니다. 입력한 변수는 매크로 창의 변수 탭에서 확인할 수 있습니다.

예) /사용 [%변수0] - 최하급 생명의 물약을 사용합니다.

매크로 수동 입력 - 인벤토리 슬롯 지정

매크로 편집 창의 매크로 명령 입력란에 '/사용 [가로번호] [세로번호]'를 정확히 입력합니다.

가로번호는 왼쪽 칸부터 1~9까지 숫자로 지정되어 있습니다.

세로번호는 윗줄부터 1~최대 인벤토리 확장 큐브까지 지정되어 있습니다.

예) /사용 [1 3] - 인벤토리 가로 1번 세로 3번째 아이템을 사용합니다.

매크로 수동 입력 - 아이템 장착

매크로 편집 창의 매크로 명령 입력란에 '/사용오른쪽 [아이템이름]' 또는 '/사용왼쪽 [아이템 이름]'을 입력하여, 오른손이나 왼손 중 원하는 곳에 아이템을 장착할 수 있습니다.

예) /사용오른쪽 [베르테론 귀고리] - 오른쪽 귀고리 슬롯에 베르테론 귀고리를 착용합니다.

예) /사용왼쪽 [베르테론 귀고리] - 왼쪽 귀고리 슬롯에 베르테론 귀고리를 착용합니다.

매크로 수동 입력 - 기타

'/지연 [시간(초)]'을 입력해 일정 시간을 지연할 수 있습니다.

지연 명령어는 최대 290초까지 사용할 수 있습니다.

예) /지연 3.5 - 3.5초 지연합니다.

'/공격'을 입력해 일반 공격을 실행할 수 있습니다.

예) /공격 - 선택한 대상을 공격합니다.

'/봉혼석 [발동/해제]'를 입력해 봉혼석 사용 상태를 변경할 수 있습니다.

예) /봉혼석 발동 - 봉혼석을 사용합니다.

'/무기셋전환'을 입력해 다른 무기로 바꿔 장착할 수 있습니다.

'/대상의대상선택'을 입력해 선택한 대상을 지원할 수 있습니다.

'/자동선택'을 입력해 공격 대상을 자동으로 선택할 수 있습니다.

매크로 주의 사항

- 매크로 안에 다른 매크로를 실행할 수 없습니다.

- 명령어 사이에 충분한 지연 시간이 필요한 경우가 있습니다.

- 스킬을 사용할 만큼의 정신력 또는 생명력이 없거나 지정한 대상이 없으면 매크로가 중지됩니다.

- 사용 예 탭에서 명령어 사용법을 확인할 수 있습니다.

- 채팅 창에서는 다음 매크로 명령어를 사용할 수 없습니다.

/지연 [%자신], [%펫], [%파티0~5], [%펫1~5], [%대상], [%이전대상], [%대상의대상], [%변수0~9]
